# 📦 Instruções de Continuidade — Projeto SynMind (Inovexa)

Este arquivo garante a continuidade perfeita do projeto SynMind/MindScan sempre que for retomado em novos chats com a IA Inovexa (GPT Inovexa / Leo Vinci).

---

## ✅ Sempre enviar ao iniciar um novo chat:

### 1. Projeto comprimido

- Nome do arquivo:
  ```
  mindscan_vX.Y.Z.zip
  ```
- Conteúdo esperado:
  - backend/ (FastAPI)
  - apps/questionnaire/ (Next.js)
  - docker-compose, .env, scripts, etc.

### 2. Prompt da MI

- Nome do arquivo:
  ```
  Prompt MI Final Integrado — SynMind.md
  ```
- Conteúdo: versão final do prompt da inteligência MI, com diretrizes técnicas e éticas.

### 3. Comando de retomada

No novo chat, digitar uma das opções:
```
Continuar SynMind
```
ou
```
Começar pelo DASS-21
```
ou
```
Começar WhatsApp
```
